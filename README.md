
## faradayawerty's dotfiles

#### description
The basics I usually need on my linux installations, an ordinary suckless workflow. Love keeping things simple minimal.

#### dependencies
* `vim`
* `tmux`
* `xcompmgr`
* `xwallpaper`
* `xclip`
* `maim`
* `hack`
* `mpv` (optional)
* `yt-dlp` (optional)
* `acpi` (optional)
* `curl` (build)
* `wget` (build)
* `gcc` (build)
* `make` (build)
* `imlib2` (build)
* `libexif2` (build)

#### installation
```
git clone https://github.com/faradayawerty/faw-dotfiles
cd faw-dotfiles
./deploy
```
